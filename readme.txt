Publish and manage advertising blocks with Google AdSense, Amazon or any custom html, in posts and sidebars using shortcodes and widgets 

== Description ==

The simple advertising tool that allows you to publish and manage advertising blocks in posts and sidebars using shortcodes and widgets. You can define ad blocks containing custom html or ads from Google AdSense, Amazon banners, etc.

= Docs & Support =

You can find [Docs](https://wp-aptools.com/adblocks/documentation/installation/) and more detailed information about Ad Blocks on [wp-aptools.com](https://wp-aptools.com). If you were unable to find the answer to your question in any of the documentation, you should check the [support forum](https://wordpress.org/support/plugin/ad-blocks/) on WordPress.org. If you can't locate any topics that pertain to your particular issue, post a new topic for it.

= Features =
*   Shortcode [adblock name="your_adblock_name"] for inserting your blocks in any posts
*   Gutenberg or classic editor mode for editing ad blocks content 
*   Auto posting: add Ad Blocks to all posts of selected types (post, page, or any custom public type) automatically
*   Widget for publishing Ad Blocks in any widget area
*   Hooks for customization (developers)

= Available in Addons =
*   Load advertising blocks content on pages via ajax for better SEO
*   Scheduling advertising by week days, start/end dates, time spans, excluded dates
*   Auto posting with include/exclude rules based on target post taxonomies
*   anything else on request

You can find [Addons](https://wp-aptools.com) and more detailed information about Ad Blocks on [wp-aptools.com](https://wp-aptools.com).

== Installation ==

1. Upload the 'ad-blocks' folder to the '/wp-content/plugins/' directory
1. Activate the Ad Blocks plugin through the 'Plugins' menu in WordPress
1. Go to Ad Blocks Settings menu to make main settings.
